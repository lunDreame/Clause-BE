# Clause (클로즈)

계약서 분석 서비스 - PDF/이미지 계약서를 업로드하면 리스크 신호를 가시화하고 쉬운 설명, 확인 질문, 협상 제안 문구를 제공합니다.

**⚠️ 중요: Clause는 법률 자문이 아니며, 정보 제공 목적입니다. 중요한 계약은 전문가 상담을 권장드립니다.**

## 주요 기능

- PDF 계약서 텍스트 추출
- 260+ 개의 룰 기반 조항 분석
- LLM 기반 리스크 분석
- PII 마스킹 및 금지어 가드
- 계약 유형별 맞춤 분석 (프리랜서, 근로, 임대차, NDA 등)

## 면책 사항

Clause는 법률 자문이 아니며, 정보 제공 목적입니다. 중요한 계약은 반드시 전문가 상담을 받으시기 바랍니다.

## 라이선스

이 프로젝트는 내부 사용 목적으로 개발되었습니다.
